Planning your day is a productive step contrary to working haphazardly. Wouldn't it be easy if you could list out your plans for the day and keep a track of them on-the-go. Planner helps you do that conveniently.


Get Started:

  • Add new items to the list by typing the task and clicking on "Add To List" (or hitting ENTER).
  • 'Check' the tasks which have been done, leaving others unchecked.
  • Create a new list (delete existing) by clicking on "Create New/Delete Existing" from Navigation Bar.
  • The Total count of tasks is displayed in the top banner.

Simple enough, isn't it?


The project has been created using AngularJS and Bootstrap frameworks along with Javascript, HTML, CSS for adding basic functionalities and enhancing user experience. Feedbacks/Suggestions for improvements are welcome.
